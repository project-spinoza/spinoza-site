Commands | Lookup Friendship

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak lookupFriendShip -i ids.txt -o frndships.txt

Arguments:

-i:

Path/Name of the file in containing id's/screen_names of the users to be looked up for frienship properties to authenticating user.

-o:

Name of the output file to which the required users will be dumped.

Description:

This command the relationships of the authenticating user to the users contained in the given input file contained of 100 twitter screen names or IDs. Values for connections can be: following, following_requested, followed_by, none, blocking, muting.
