Commands | Dump Mutes List

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpMutesList -limit 1 -o mutes.txt

Arguments:

-o:

Name of the output file to which muted users will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

This command dumps all users that are muted by the authenticating twitter user to the given output file. The number of retrieved twitter users are subjected to Twitter rate limits.
