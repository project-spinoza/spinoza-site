Dump by Twitter Screen Name:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpFollowerIDs -uname ScreenName -limit 1 -o output.csv

Arguments:

-uname:

Target User Twitter ScreenName.

-o:

Name of the output file to which the target user follower will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

This command dumps a twitter user's followers IDs, screen name, followers count, and friends count to the given output file. The number of followers are subjected to Twitter rate limits.




Dump by Twitter ID:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpFollowerIDs -uid 101010111 -limit 1 -o output.csv

-uid:

Target User's Twitter ID.

-o:

Name of the output file to which the target user follower will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

Based on provided credentials, this command dumps a valid twitter user's followers ids, screen name, followers count, and friends count to a file. The number of followers are subjected to Twitter rate limits.
