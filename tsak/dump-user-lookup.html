Commands | Dump User Lookup

Dump by Twitter Screen Name:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpUsersLookup -i ids.txt -o lookup.txt

Arguments:

-i:

Path/Name of the file in containing id's/screen_names of the users to be Looked up.

-o:

Name of the output file to which the required users will be dumped.

Description:

This command dumps the specified twitter users data whose twitter's ids/screen_name are given in the input file.
