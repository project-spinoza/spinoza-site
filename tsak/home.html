Welcome to the Twitter Swiss Army Knife(TSAK)!

The purpose of this tool is to create a simple command line utility that will allow users to interact with Twitter's public API. TSAK is an open source project that enables end users to use Twitter APIs without going into in-depth Twitter technicalities. All the end user has to do is to provide Twitter credentials (App + User keys) and an action query to pull the information that the user wants to access. TSAK will work its magic and take care of the rest.
