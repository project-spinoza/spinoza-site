Dump by Twitter Screen Name:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpFriendIDs -uname ScreenName -limit 1 -o friendsIDs.txt

Arguments:

-uname:

Target User's Twitter ScreenName.

-o:

Name of the output file to which the target user's friends will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

This command dumps a valid twitter user's friends IDs, screen name, followers count, and friends count to the given output file. The number of friends IDs are subjected to Twitter rate limits.
