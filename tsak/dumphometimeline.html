Dump Home Timeline:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpHomeTimeLine -o output.txt

Arguments:

-o:

Name of the output file to which Home timeline will be dumped.

Description:

This command dumps Home TimeLine (Based on provided Twitter Keys and access token) to the provided output file. The number of dumped Tweets are subjected to Twitter rate limits.
