Commands | Stream statuses

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak streamStatuses -keywords "iPhone" -store true -o streamsOut.txt

Arguments:

-o:

Name of the output file, optional parameter

-keywords:

Twitter Streaming tweets contain these keywords

-store:

A boolean parameter which represent stream storage to a file

Description:

Based on provided credentials, this command will dump all the streaming tweets.
