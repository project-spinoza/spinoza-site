Commands | Dump User Suggestions

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpUserSuggestions -slug family -o suggestions.txt

Arguments:

-slug:

The short name of list or a category.

-o:

Name of the output file to which the result users will be dumped.

Description:

This command dumps the users in a specified twitter category to the given output file.
