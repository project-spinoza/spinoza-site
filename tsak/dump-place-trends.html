Commands | Dump Place Trends

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpPlaceTrends -woeid 010101 -o trends.txt

Arguments:

-woeid :

The Yahoo! Where On Earth ID of the location.

-o:

Name of output file to which the trends will be dumped.

Description:

This command dumps top 10 trending topics for the specific WOEID to the given output file, if trending information is available for it.
