Commands | Dump Account Setting

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpAccountSettings -limit 1 -o accountSettings.txt

Arguments:

-o:

Name of the output file to which the result will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

Based on provided credentials, this command will dump all of the account properties and settings the authenticating user to provided output file.
