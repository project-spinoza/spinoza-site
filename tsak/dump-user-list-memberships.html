Dump by Twitter Screen Name:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpUserListMemberships -uname screenName -limit 1 -o memberships.txt

Arguments:

-uname:

Screen name of the user who has membership in twitter lists.

-o:

Name of the output file to which the twitter Lists will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

This command dumps twitter lists to the given output file, in which the specified user has been added. The number of the retrieved twitter lists is subjected to twitter rate limit status.




Dump by Twitter ID:

FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

tsak dumpUserListMemberships -uid 010101 -limit 1 -o memberships.txt

-uid:

Numeric ID of the user who has membership in twitter lists.

-o:

Name of the output file to which the List Memebership users will be dumped.

-limit:

Represents API limits for the above command and it has a default value 1. This argument is optional.

Description:

This command dumps twitter lists to the given output file, in which the specified user has been added. The number of the retrieved twitter lists is subjected to twitter rate limit status.
