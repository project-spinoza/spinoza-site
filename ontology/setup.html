Ontology is an open source project, containing the following categories.

  • Car
  • Data science
  • Lifestyle
  • Newspaper
  • Politics
  • Sports team

By navigating to the Ontology site the user can download the data in different formats. For example

  • CSV (Comma Separated Values)
  • JSON (JavaScript Object Notation)
  • XML (Extensible Markup Language)
  • SQL (Structured Query Language)
After downloading the data in one of the above format, there will be a lot of information. Like Title, Url of image, Tag, body(description), Url of Homepage, Url of Twitter, Url of Wikipedia, Url of Pinterest, Url of Google Plus, Url of Linkedin, Url of Foursquare, Url of Forums, Url of Other Links and Parent.
