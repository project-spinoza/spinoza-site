Ontology is an open source project. It contains the following categories.

  • Car

    • It contains Car related data. Like different brands of car, names of car etc.

  • Data science

    • It contains Data science related data. Like Elastic Map Reduce, Graph Databases, Information visualization and NoSql databses.

  • Lifestyle

    • It contains Lifestyle related data. Like Cinemas, Clothing, Eateries, Eyewear, Gyms, Shoes, Shopping mall, Soft Drinks, Sweets, Watches, Wines and Jewelry.

  • Newspaper

    • It contains Newspaper related data. Like Blog, Magazines, Newspaper and Tabloid.

  • Politics

    • It contains Politics related data. Like Democratic Party, Other Governors, Other Senators and Republican.

  • Sports team

    • It contains Sports related data. Like American football, Badminton, Baseball, Basketball, Cricket, Field hockey, Golf, Ice Hockey, Rugby football, Soccer, Table Tennis, Tennis and VolleyBall.
