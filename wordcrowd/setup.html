WordCrowd

WordCrowd is a javaScript implementation of WordCram.

Step 1:

Download zip file of WordCrowd.
OR

Clone it using the following command.

git clone https://github.com/project-spinoza/WordCrowd.git

Step 2:

Navigate to the examples directory and run the index.html file in web browser.

Step 3: Running WordCrowd

Running WordCrowd

var settings = {
"container":"body",
"data":"Input text data for visualization"
};
var wordCrowd = new WordCrowd(settings);
